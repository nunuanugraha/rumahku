# 🏠 RumahKu - House Management System

![Status](https://img.shields.io/badge/Status-Active-success?style=for-the-badge)
![Version](https://img.shields.io/badge/Version-1.0.0-blue?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-orange?style=for-the-badge)
![PWA](https://img.shields.io/badge/PWA-Supported-purple?style=for-the-badge)

**RumahKu** adalah aplikasi manajemen kebutuhan rumah tangga modern yang dirancang untuk membantu keluarga mengelola daftar belanja, prioritas barang, dan anggaran bulanan secara real-time. Dibangun dengan fokus pada kecepatan (Performance) dan kemudahan penggunaan (UX).

---

## ✨ Fitur Unggulan

- 📊 **Dashboard Ringkasan**: Pantau total pengeluaran dan progres belanja dalam satu layar.
- ✅ **Manajemen Belanja (CRUD)**: Tambah, edit, dan hapus kebutuhan rumah tangga dengan mudah.
- 💰 **Budgeting System**: Kelola anggaran bulanan dan pantau sisa biaya secara otomatis.
- 🌙 **Dark Mode Pro**: Antarmuka adaptif yang nyaman di mata untuk penggunaan siang maupun malam.
- 📱 **Fully Responsive**: Optimal di perangkat Mac, Windows, iPad, hingga Android/iPhone.
- 🚀 **PWA Ready**: Dapat di-install langsung ke layar utama tanpa melalui App Store/Play Store.

---

## 🛠️ Stack Teknologi

Aplikasi ini dibangun menggunakan pendekatan **Clean Code** dan tanpa ketergantungan library eksternal yang berat (Vanilla Scripting):

- **Frontend**: HTML5, Modern CSS3 (Variables & Grid)
- **Icons**: Google Material Symbols Rounded
- **Typography**: DM Sans & DM Serif Display
- **Scripting**: Vanilla JavaScript (ES6+)
- **Storage**: LocalStorage (Offline Persistence)
